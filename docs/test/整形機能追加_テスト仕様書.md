# [Variableインスタンスの整形機能追加] 単体テスト仕様書

## 1. テスト目的

`Variable` インスタンスの整形(reshape)が正しく行えているかを確認
## 2. テスト対象クラス・メソッド

| 対象クラス | 対象メソッド | 確認内容の要約 |
| --- | --- | --- |
| `Variable` | `reshape` | reshapeが正しく行えているか確認 |

#### 3.1 正常系テスト (Normal Cases) - 追加分

| ID | テストケース名 | 入力値 (Input) | 期待される結果 (Expected) | 検証方法 |
| --- | --- | --- | --- | --- |
| N-01 | Reshape: 形状変更 | `x = F.Variable(np.array([[2, 3], [4, 5], [6, 7]]))` | `y.shape == (3, 2)` | `assertEqual` |
| N-02 | Transpose: 軸指定 | `x = F.Variable(np.array([[2, 3], [4, 5], [6, 7]]))`, `axes=(1, 0)` | `y.shape == (3, 2)` | `assertEqual` |
| N-03 | Reshape: 自動計算 | `x = F.Variable(np.array([[2, 3], [4, 5], [6, 7]]))`, `shape=(-1,)` | `y.shape == (6,)` | `assertEqual` |
| N-04 | Transpose: 勾配の形状 | ` x = F.Variable(np.array([[2, 3], [4, 5], [6, 7]]))` | `x.grad.shape == (3, 2)` | `assertEqual` |
| N-05 | Reshape: 勾配の形状 | ` x = F.Variable(np.array([[2, 3], [4, 5], [6, 7]]))` | `x.grad.shape == (3, 2)` | `assertEqual` |

#### 3.2 境界値・異常系テスト (Edge/Exception Cases) - 追加分

| ID | テストケース名 | 入力値 (Input) | 期待される結果 (Expected) | 備考 |
| --- | --- | --- | --- | --- |
| E-01 | Reshape: 要素数不一致 | `x = F.Variable(np.array([[2, 3], [4, 5], [6, 7]]))`, `shape=(5,)` | `ValueError` が発生すること |  |
| E-02 | Transpose: 同一形状 | `x = F.Variable(np.array([[2, 3], [4, 5], [6, 7]]))` | `ValueError` が発生すること | 入力値に存在しない軸を指定 |
