# [sin関数] 設計仕様書

## 1. 概要
numpyを使用せず、テイラー展開を用いた近似を行い活性化関数として使用する。
* **モジュール**: `core_simple.py`モジュールに追加
## 2. メソッド/関数 
| 使用モジュール | 使用メソッド | 用途 |
| :--- | :--- | :--- |
| `math` | `factorial`| 階乗を行うために使用 |

### プロトタイプ 
`sin(x, threshhold=0.0001)`
- **引数(x)**: ndarray型変数
- **引数(threshhold=0.0001)**: しきい値の設定。値が小さいほど制度が向上する
- **戻り値**: `y`

**変数名一覧**
| 変数名 | 説明 | 使用箇所 | 補足 |
| :--- | :--- | :--- | :--- |
| `y` | 計算結果を返す際に使用 | forループ内部 | forループ前に0で初期化を行う|
| `i` | ループカウンタ | forループ内部 | カウンタとして階乗内(`2 * i + 1`)などで使用 |
| `c` | `factorial`の計算結果を格納 | forループ内部 |  |
| `t` | `c * x ** (2 * i + 1)`の計算結果を格納 | forループ内部 | forループ内でyに加算していく |

#### 処理フロー

1. **初期化**: `y = 0`
2. **ループ実行**: [`for i in range(10000)`]
    - 2.1. `c`に`(-1) * i / math.factorial(2 * i +1)`を代入
    - 2.2. `t`に `c * x ** (2 * i + 1)` を代入
    - 2.3. `y`に`t`の結果を加算[`y = y + t`]
3. **ループの終了条件**
    - tの値がしきい値より小さい場合処理を停止
        - `if abs(t.data) < threshhold: break`
